Mta Data Feeds

The MTA provides free API services for anyone to figure out why their train is delayed. Through their developer resources page on their site, they offer many different types of APIs for programmers to mess around with, for FREE. In this little project, I will be using the MTA’s services alerts feed API to display the latest subway service changes on a web page. This page will use vanilla javascript functions to fetch data.
