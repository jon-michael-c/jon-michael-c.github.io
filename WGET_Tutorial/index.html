wget and cURL

Intro

WGET is a simple command used on the terminal to collect files from the internet, mainly integrated within GNU/Linux operating systems. It works by making request to the internet and pulls back requested items to be downloaded. This could be anything from a file, picture, or spreadsheet. CURL works similiarly to the WGET in pulling data off the from internet. However, cURL is powered by the libcurl

Windows

curl.exe -O "https://testsite.com/bigdata.csv

Linux

curl -O "https://testsite.com/bigdata.csv
